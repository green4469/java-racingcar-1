###기능 구현 목록
*볼드는 테스트 코드 작성*
- [x] 이름 입력
    - [x] 예외 처리
- [x] 자동차 생성 
    - [x] 레이스 객체에 자동차 배열 저장
- [x] 횟수 입력
    - [x] 예외 처리
    - [x] 레이스 객체에 횟수 저장
- [ ] 레이싱 진행
    - [x] **난수 발생**
    - [x] **자동차 전진 기능**
    - [ ] 횟수 만큼 반복
    - [ ] 과정 출력
    - [ ] **우승자 판별**
- [ ] 우승자 출력
- [ ] 리팩토링
